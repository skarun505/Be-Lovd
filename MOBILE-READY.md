  - You click a menu item

#### 2. **Responsive Breakpoints**
- **Desktop** (1024px+) - Full layout with all features
- **Tablet** (768px-1024px) - Optimized spacing and sizing
- **Mobile Landscape** (481px-768px) - Single column, stacked layout
- **Mobile Portrait** (360px-480px) - Touch-optimized buttons
- **Small Phones** (under 360px) - Extra small spacing

#### 3. **Mobile-Specific Optimizations**

**Navigation:**
- Logo scales down on mobile (1.4rem)
- Hamburger menu replaces horizontal menu
- Larger touch targets (44px minimum)

**Hero Sections:**
- Images appear ABOVE text on mobile
- Font sizes scale appropriately:
  - Desktop: 6rem titles
  - Tablet: 4.5rem
  - Mobile: 2.8rem
  - Small phones: 2.2rem
- Floating elements reduced in size
- Product images scale: 500px → 280px → 240px → 200px

**Product Cards:**
- Switch to single column on mobile
- Image height adjusts: 300px → 250px → 200px
- Cards stay touch-friendly with proper spacing

**Buttons:**
- All buttons have minimum 44px touch target
- Optimized padding for mobile taps
- No hover effects on touch devices (prevents sticky state)

**Footer:**
- Stacks into single column
- Newsletter form goes vertical
- Social icons center-aligned
- Full-width submit button

#### 4. **Performance Features**
✅ **Lazy Loading** - Images load as you scroll  
✅ **Optimized Images** - Proper sizing from CDN  
✅ **Touch Optimizations** - 44px minimum tap targets
