## HOW TO USE THE WEBSITE

### For Customers:

**Step 1: Browse Products**
- Open `index.html`
- Scroll to "Our Valentine's Collection"
- View 9 premium cakes

**Step 2: Add to Cart**
- Click "Add to Cart" on desired cakes
- See confirmation "Added to cart!"
- Cart count updates in navigation

**Step 3: View Cart**
- Click cart icon in top right
- Opens `cart.html` page
- View all added items
- Adjust quantities with +/- buttons
- Apply coupon codes if available
- See delivery charges and total

**Step 4: Checkout**
- Click "Proceed to Checkout"
- Opens `checkout.html` page
- Fill delivery information
- Select delivery date & time
- Choose payment method
- Review order summary
- Click "Place Order"

**Step 5: Confirmation**
- Redirected to `order-confirmation.html`
- View order number
- Print receipt if needed
- Receive confirmation details
